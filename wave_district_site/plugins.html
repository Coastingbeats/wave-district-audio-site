Wave District Channel Strip

Wave District Audio

Channel Strip

A 4-slot channel strip — Gate, 7-Band EQ, and dual independent compressors (Comp A/B) with switchable classic emulations (1176, LA-2A, Distressor, SSL Bus, SSL Channel, dbx 160, API 2500). Includes an inline de-esser, SSL/Transparent/API/Neve EQ character modes, and a real-time spectrum analyzer with drag-to-tune band nodes. Resizable UI, mono/stereo support in Logic.
